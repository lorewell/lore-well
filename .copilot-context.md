## 项目定位

React 19 + Vite 8 + Phaser 4 + Zustand 5 + Tailwind CSS v4 的网页 RPG。
**无行走、地点切换式**回合制游戏。React 负责所有 UI，Phaser 负责背景渲染，Zustand 管理全游戏状态。

## 技术栈

| 技术 | 版本 | 用途 |
|---|---|---|
| React + TypeScript | 19 / ~6.0.2 | 页面框架 + 所有 UI |
| Vite | 8 | 构建工具 |
| Phaser | 4.0.0 | 场景背景渲染（npm 包） |
| Zustand | 5.0.12 | 全局状态 + localStorage 持久化 |
| React Router | v7.14.2 | 路由（`/` 主菜单 / `/game` 游戏） |
| Tailwind CSS | v4.2.4 | 样式（`@import "tailwindcss"` in index.css） |

## 重要设计约定

- **双属性字段**：`character.baseStats` = 纯基础值；`character.stats` = `applyEquipmentBonuses(baseStats, equipment)`，装备/升级时重算
- **商店触发**：对话节点 `next='shop'` → `DialogBox` 调 `openShop(npcId)` → store 存 `activeShopNpcId` → `GameScreen` 渲染 `ShopPanel`
- **任务自动触发**：`QuestObjective.trigger` 声明条件，各动作调 `_autoCompleteObjectives` 自动勾选，全完成则自动调 `completeQuest` 发奖励
- **战斗失败**：`respawnAtVillage()` 传送至 village，HP/MP 恢复 50%，装备道具保留
- **buyItem 签名**：`buyItem(itemId: string, price: number): boolean`（price 从 ShopEntry 直接传入）
- **sellItem 签名**：`sellItem(itemId: string, price: number): boolean`
- **零 TS 错误原则**：每次修改后必须通过 `get_errors` 验证，不允许遗留错误
