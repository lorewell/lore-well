## 任务系统说明

### 任务状态

| 状态 | 含义 | 任务日志显示 |
|------|------|------------|
| `locked` | 尚未触发，条件未达成 | 显示为"未解锁" |
| `active` | 进行中 | 显示在"进行中"分组 |
| `completed` | 所有目标完成，奖励已领取 | 显示在"已完成"分组 |

### 自动触发机制

目标通过 `trigger` 字段声明触发条件，游戏状态变化时**自动**勾选，无需手动交互：

| 触发器类型 | 触发时机 |
|-----------|---------|
| `have_item` | 背包中出现对应物品时（拾取/战斗掉落） |
| `defeat_enemy` | 战斗胜利，击败对应敌人时 |
| `visit_location` | 前往对应地点时 |
| `talk_npc` | 开启与对应 NPC 的对话时 |

所有目标完成后，任务**自动完成**并发放奖励（无需额外操作）。

### 奖励类型

| 奖励 | 说明 |
|------|------|
| EXP | 立即加到角色经验，可能触发升级 |
| 金币 | 加入当前金币储量 |
| 物品 | 直接放入背包 |

## 主线任务

### 长老的委托

```
ID：quest_elder
类型：主线
初始状态：locked → 对话长老后激活
```

**背景故事**

晨曦村近来笼罩在不明的阴影之下，牲畜失踪、夜间有异声从远方传来。长老艾尔文是村中唯一了解古代神殿历史的人，他察觉到遗迹之中封印已有松动的迹象。由于年迈无法亲行，他将这一重担寄托于偶然路过的旅行者——你。

**解锁方式**

前往**晨曦村**，进入**长老之屋**与**长老艾尔文**对话（任务自动激活）。

**任务目标**

| 步骤 | 目标描述 | 触发条件 |
|------|---------|---------|
| 1 | 与长老艾尔文交谈 | 开启长老对话 |
| 2 | 在森林深处找到古代钥匙 | 背包中获得 `ancient_key` |
| 3 | 前往古代神殿遗迹 | 到达 `temple_ruins` |

**推荐路线**

```
晨曦村（与长老对话）
  └─▶ 幽暗森林（穿越）
        └─▶ 森林深处（拾取古代钥匙·石台上的匣子）
              └─▶ 幽暗森林（返回）
                    └─▶ 晨曦村
                          └─▶ 古代神殿遗迹（任务完成）
```

**任务奖励**

| 奖励 | 数量 |
|------|------|
| 经验值 | +200 EXP |
| 金币 | +100 G |

**后续钩子（待实现）**  
到达神殿后，神秘祭坛仍等待进一步探索。古代钥匙的真正用途尚未揭晓——封印究竟通向何处？

## 支线任务

### 森林的威胁

```
ID：quest_forest
类型：支线（战斗向）
初始状态：locked → 对话长老后激活
```

**背景故事**

幽暗森林的生态平衡正在被破坏。哥布林族群异常活跃，开始袭击试图进入森林的商队；森林狼群也受到某种力量驱使，变得格外凶猛。村民们已多日不敢外出，农田放弃收割，粮食供应岌岌可危。

**解锁方式**

与**长老艾尔文**对话后自动激活（与主线任务同步触发）。

**任务目标**

| 步骤 | 目标描述 | 触发条件 |
|------|---------|---------|
| 1 | 击败哥布林 | 战斗胜利：`goblin` |
| 2 | 击败森林狼 | 战斗胜利：`forest_wolf` |

**推荐地点**

- 哥布林：**幽暗森林**（哥布林巡逻队）
- 森林狼：**幽暗森林**（森林狼群）或**森林深处**（古老的狼灵，难度更高）

**任务奖励**

| 奖励 | 数量 |
|------|------|
| 经验值 | +150 EXP |
| 金币 | +60 G |

**提示**  
建议先在晨曦村击败几只史莱姆练级（获取 20 EXP），或购买/拾取生命药水再进入森林。

### 补给短缺

```
ID：quest_supplies
类型：支线（新手引导）
初始状态：active（游戏开始即激活）
```

**背景故事**

独自踏上旅途的冒险者，最基本的生存法则就是备足物资。一瓶生命药水，往往是生死之间的分水岭。

**解锁方式**

游戏开始时**默认激活**，无需任何触发。

**任务目标**

| 步骤 | 目标描述 | 触发条件 |
|------|---------|---------|
| 1 | 获得至少 1 瓶生命药水 | 背包中出现 `health_potion` |

**获取方式**

| 来源 | 说明 |
|------|------|
| 战斗掉落 | 史莱姆（40% 概率）、哥布林（30% 概率） |
| 地点拾取 | 幽暗森林·废弃的行囊（一次性） |

**任务奖励**

| 奖励 | 数量 |
|------|------|
| 经验值 | +50 EXP |

### 铁匠委托

```
ID：quest_blacksmith
类型：支线（收集向）
初始状态：locked → 首次与铁匠托尔对话后激活
```

**背景故事**

铁匠托尔是晨曦村唯一的武器制造者，近来原材料短缺让他无法继续锻造。废弃矿洞中堆积着大量未采集的铁矿石，但巢穴内的洞穴蜘蛛让矿工们望而却步。托尔希望有人能替他深入矿洞，带回足够的铁矿石——作为报酬，他会为你打造一把上等的精钢剑。

**解锁方式**

前往**晨曦村·铁匠铺**，与**铁匠托尔**进行首次对话后自动激活。

**任务目标**

| 步骤 | 目标描述 | 触发条件 |
|------|---------|----------|
| 1 | 收集铁矿石 | 背包中出现 `iron_ore` |
| 2 | 将铁矿石交给铁匠托尔 | 再次与托尔对话 |

**推荐路线**

```
晨曦村（与托尔对话，激活任务）
  └─▶ 晨曦村南门
        └─▶ 废弃矿洞（矿洞入口）
              └─▶ 矿洞深处
                    └─▶ 矿石矿脉（击败洞穴蜘蛛，50% 掉落铁矿石）
              └─▶ 返回晨曦村
                    └─▶ 铁匠铺（交给托尔，任务完成）
```

**任务奖励**

| 奖励 | 内容 |
|------|------|
| 经验值 | +100 EXP |
| 物品 | 精钢剑（`steel_sword`，ATK+20，不可购买的专属奖励） |

**提示**  
矿洞蜘蛛速度极快（SPD 18），建议提前装备迅捷戒指或使用技能先手控场。矿石矿脉节点的掉落率最高（50%），优先前往深处挖矿效率最佳。

## 扩展任务设计指引

新增任务只需在 `src/data/quests.ts` 的 `INITIAL_QUESTS` 数组中追加一个 `Quest` 对象。

**示例模板**

```typescript
{
  id: 'quest_new',
  title: '任务标题',
  description: '任务背景描述文字',
  status: 'locked',           // 或 'active'（立即激活）
  objectives: [
    {
      id: 'obj_1',
      description: '目标描述',
      completed: false,
      trigger: { type: 'defeat_enemy', enemyId: 'goblin' }
      // 可用触发器：have_item / defeat_enemy / visit_location / talk_npc
    },
  ],
  reward: { exp: 100, gold: 50 },
}
```

**触发器速查**

```typescript
{ type: 'have_item',      itemId: 'health_potion' }
{ type: 'defeat_enemy',   enemyId: 'goblin' }
{ type: 'visit_location', locationId: 'temple_ruins' }
{ type: 'talk_npc',       npcId: 'elder' }
```

**任务激活方式**  
- 默认 `status: 'locked'`，由代码主动调用 `store.activateQuest(questId)` 解锁  
- 目前长老对话触发 `quest_elder` 与 `quest_forest`（在 `gameStore.ts > openDialogue` 中处理）
- 新的触发点可在 `LocationPanel`、`DialogBox` 或 `CombatPanel` 中加入 `activateQuest` 调用
