## 物品系统说明

### 物品类型

| 类型 | 说明 | 背包显示颜色 |
|------|------|------------|
| `consumable` | 消耗品，使用后消耗 1 个，可在战斗内或地图界面使用 | 绿色 |
| `equipment` | 装备，可穿戴到对应槽位，提供属性加成 | 蓝色 |
| `quest` | 任务道具，不可使用，完成任务自动消耗或锁定 | 橙色 |
| `misc` | 杂项，堆叠存储，可用于任务提交 | 灰色 |

### 堆叠规则

- 消耗品与杂项均可堆叠（`stackable: true`）
- 装备不可堆叠，同类型可多件（仅穿戴一件生效）
- 任务道具不可堆叠，多次获取会被忽略

## 装备系统详解

### 装备槽位

| 槽位 | 可装备物品举例 |
|------|-------------|
| `weapon` | 铁剑、精钢剑、法术手杖 |
| `armor` | 皮甲、锁子甲 |
| `accessory` | 迅捷戒指（+SPD）、活力护符（+maxHP） |

每个槽位同时只能装备一件物品。装备/卸装后属性立即重算：

```
有效属性 = baseStats（等级决定）+ Σ 已装备物品的 statBonus
```

饰品可加成 `maxHp` / `maxMp`，这些上限变化会同步影响当前 HP/MP 的百分比换算。

### 装备建议

| 阶段 | 推荐配置 |
|------|---------|
| 早期（LV1-2） | 铁剑 + 皮甲 |
| 中期（LV3-4） | 铁剑/法术手杖 + 皮甲 + 活力护符 |
| 后期（LV5+） | 精钢剑 + 锁子甲 + 迅捷戒指 |
