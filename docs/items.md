## 物品系统说明

### 物品类型

| 类型 | 说明 | 背包显示颜色 |
|------|------|------------|
| `consumable` | 消耗品，使用后消耗 1 个，可在战斗内或地图界面使用 | 绿色 |
| `equipment` | 装备，可穿戴到对应槽位，提升角色属性 | 蓝色 |
| `quest` | 任务道具，不可使用，仅用于任务触发 | 灰色 |

### 装备槽位

| 槽位 | 类型 | 当前可装备物品 |
|------|------|--------------|
| `weapon` | 武器 | 铁剑 |
| `armor` | 护甲 | 皮甲 |
| `accessory` | 饰品 | ——（暂无） |

### 堆叠规则

- `stackable: true`：同类物品叠加计数显示（如药水 ×3）
- `stackable: false`：每件单独占一格（武器、护甲、任务道具）

## 装备

### 铁剑

```
ID：iron_sword
类型：equipment（装备）
装备槽：weapon（武器）
可叠加：否
```

| 属性加成 | 数值 |
|---------|------|
| ATK | +10 |

**外观描述**  
一把普通的铁剑，锋利而耐用。适合初出茅庐的冒险者，是当前游戏中唯一可获得的武器。

**获取方式**

| 来源 | 说明 | 概率/条件 |
|------|------|----------|
| 哥布林 | 战斗掉落 | 10% |

**属性影响**  
装备后 ATK 从基础值（20）提升至 30，对哥布林（DEF 5）的普通攻击期望伤害提升约 10 点。

### 古代钥匙

```
ID：ancient_key
```

## 装备系统详解

### 属性计算

```
有效属性 = 基础属性（baseStats）+ 所有已装备物品的 statBonus 之和
```

装备后属性**立即生效**，当前 HP/MP 按比例换算（避免装护甲导致满血变满血）。

### 同槽替换

装备新物品时，若该槽位已有装备，旧装备会**自动回到背包**——不会丢失。

### 卸装

在背包面板左侧装备栏中点击 **−** 按钮，装备归还背包，属性即时恢复为基础值。

## 扩展物品设计指引

```typescript
// src/data/items.ts — 添加新物品示例

magic_staff: {
  id: 'magic_staff',
  name: '法术手杖',
  description: '蕴含魔力的手杖，可增强法术效果。',
  type: 'equipment',
  equipSlot: 'weapon',
  statBonus: { atk: 5, maxMp: 20 },  // 多属性加成
  stackable: false,
},

super_potion: {
  id: 'super_potion',
  name: '高级药水',
  description: '恢复 120 点 HP。',
  type: 'consumable',
  effect: { hp: 120 },
  stackable: true,
},
```

> 新物品添加后，在 `src/data/enemies.ts` 的 `dropTable` 或 `src/data/locations.ts` 的 `interactions` 中引用即可自动出现在游戏中。
