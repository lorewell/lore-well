## NPC 人物

### 老板娘 玛格

```
位置：晨曦村 — 暮光客栈
ID：innkeeper
```

**背景**  
暮光客栈的经营者，是晨曦村中最消息灵通的人物之一。为人热情，总是能给旅行者提供信息和休憩之所。她对村子近来的异常深感不安，但仍尽力维持客栈的日常运转。

**基础信息**

| 属性 | 内容 |
|------|------|
| 称呼 | 老板娘 玛格（Marg） |
| 职业 | 客栈经营者 |
| 所在地 | 晨曦村·暮光客栈 |
| 可交互 | 是（对话 + 休息） |

**对话分支**

| 选项 | 效果 |
|------|------|
| 我想休息 | 立即回复玩家全部 HP / MP |
| 最近有什么消息吗？ | 揭露森林异常信息（世界观背景） |
| 不了，谢谢 | 关闭对话 |

**相关人物**  
- 与**铁匠托尔**同住一村，互相熟识  
- 间接了解**长老艾尔文**的忧虑

### 铁匠 托尔

```
位置：晨曦村 — 铁匠铺
ID：blacksmith
```

**背景**  
村中唯一的铁匠，擅长打造武器和护甲。性格直来直往，不爱与外人多话。他需要铁矿石才能为玩家打造装备，是装备强化系统的关键 NPC（功能尚未完全实现，有待扩展）。

**基础信息**

| 属性 | 内容 |
|------|------|
| 称呼 | 铁匠 托尔（Thor） |
| 职业 | 铁匠、装备锻造师 |
| 所在地 | 晨曦村·铁匠铺 |
| 可交互 | 是（对话，铸造功能待完善） |

**对话分支**

| 选项 | 效果 |
|------|------|
| 我有铁矿石 | 进入铸造对话（功能占位） |
| 我去找找看 | 关闭对话 |

**相关人物**  
- 潜在的装备提供者，与玩家构成「资源换装备」关系  
- 在世界观中与**长老艾尔文**关系中立，关注实际利益

> **扩展方向**：添加铁矿石道具、完善铸造/强化 UI，可解锁玩家特殊武器支线。

### 长老 艾尔文

```
位置：晨曦村 — 长老之屋
ID：elder
```

**背景**  
晨曦村的精神领袖，也是神殿遗迹历史的最后守护者。年迈却眼光敏锐，他是察觉到"黑暗蔓延"危机的第一人。与玩家对话后，会触发主线任务**「长老的委托」**，并赠予古代钥匙。

**基础信息**

| 属性 | 内容 |
|------|------|
| 称呼 | 长老 艾尔文（Erwin） |
| 职业 | 村庄长老、知识守护者 |
| 所在地 | 晨曦村·长老之屋 |
| 可交互 | 是（对话，触发主线） |

**对话分支**

| 选项 | 效果 |
|------|------|
| 告诉我更多 | 获得主线背景信息 |
| 我还没准备好 | 关闭对话 |

**触发事件**（对话后自动触发）  
- 激活任务：**长老的委托**（`quest_elder`）  
- 激活任务：**森林的威胁**（`quest_forest`）

**相关人物**  
- 主线任务委托人，与**玩家**构成主从委托关系  
- 了解**古代神殿遗迹**的历史，神秘祭坛与他有关联  
- 间接催动**铁匠托尔**为玩家服务（背后权威）
