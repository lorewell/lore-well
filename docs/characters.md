### 初始技能

| 技能名 | MP 消耗 | 效果 |

| 属性 | 每级增长 |
|------|---------|
| 最大 HP | +20 |

> **装备机制**：基础属性（`baseStats`）不含装备加成，有效属性（`stats`）= baseStats + 所有已装备物品加成（含 maxHp / maxMp），每次装备/卸装/升级后重新计算。

## NPC 人物

### 隐秘瀑布

```
位置：晨曦村·村子北方（village_north 节点）
ID：waterfall
```

**背景**
一道从山间石缝倾泻而下的银白瀑布，水声如鼓，雾气弥漫。这里是长老艾尔文发现昏迷旅行者的地方。凝视瀑布，玩家可以触发破碎的记忆片段——黑色的尖塔、漫天战火和模糊的面孔。

**对话分支**

| 选项 | 效果 |
|------|------|
| 凝视瀑布，尝试回忆 | 触发记忆碎片节点 |
| 没事了 | 关闭对话 |

---

### 老板娘 玛格

```
位置：晨曦村·暮光客栈（village_west 节点）
ID：innkeeper
```

**背景**
暮光客栈的经营者，是晨曦村中最消息灵通的人物之一。为人热情，总能给旅行者提供信息和休憩之所。她对村子近来的异常深感不安，但仍尽力维持客栈的日常运转。

**基础信息**

| 属性 | 内容 |
|------|------|
| 称呼 | 老板娘 玛格（Marg） |
| 职业 | 客栈经营者 |
| 所在地 | 晨曦村·暮光客栈 |
| 可交互 | 是（对话 + 休息 + 商店） |

**对话分支**

| 选项 | 效果 |
|------|------|
| 我想休息 | 立即回复玩家全部 HP / MP |
| 我想买些物资 | 打开客栈小卖部（生命药水等） |
| 最近有什么消息吗？ | 揭露矿洞异常和神殿奇光 |
| 没事了 | 关闭对话 |

---

### 铁匠 托尔

```
位置：晨曦村·铁匠铺（village_east 节点）
ID：blacksmith
```

**背景**
村中唯一的铁匠，擅长打造武器和护甲。性格直来直往，不爱与外人多话。他需要铁矿石才能为玩家打造装备，是铁匠委托任务的委托人，也是武器装备的主要商店来源。

**基础信息**

| 属性 | 内容 |
|------|------|
| 称呼 | 铁匠 托尔（Thor） |
| 职业 | 铁匠、装备锻造师 |
| 所在地 | 晨曦村·铁匠铺 |
| 可交互 | 是（对话 + 商店） |

**对话分支**

| 选项 | 效果 |
|------|------|
| 我想看看你的商品 | 打开铁匠铺（武器/护甲/饰品） |
| 我有铁矿石 | 进入铸造对话（任务相关） |
| 没事了 | 关闭对话 |

**触发事件**（首次对话）
- 激活任务：**铁匠委托**（`quest_blacksmith`）

### 长老 艾尔文

```
位置：晨曦村·村长之屋（village_west 节点）
ID：elder
```

**背景**
晨曦村的精神领袖，也是神殿遗迹历史的最后守护者。年迈却眼光敏锐，是察觉"黑暗蔓延"危机的第一人。与玩家对话后，会触发主线任务**「长老的委托」**，并赠予古代钥匙。

**基础信息**

| 属性 | 内容 |
|------|------|
| 称呼 | 长老 艾尔文（Erwin） |
| 职业 | 村庄长老、知识守护者 |
| 所在地 | 晨曦村·村长之屋 |
| 可交互 | 是（对话，触发主线） |

**对话分支**

| 选项 | 效果 |
|------|------|
| 我的记忆全部消失了 | 进入失忆背景节点 |
| 村子现在还安全吗？ | 揭露危险信息，可推进到委托节点 |
| 谢谢你救了我 | 感谢对话节点 |
| 告诉我更多关于神殿的事 | 获得古代钥匙（主线关键道具） |
| 没事了 | 关闭对话 |

**触发事件**（对话后自动触发）
- 激活任务：**长老的委托**（`quest_elder`）
- 激活任务：**森林的威胁**（`quest_forest`）

### 村长 格雷

```
位置：晨曦村·村长之屋（village_west 节点）
ID：village_chief
```

**背景**
落瀑村的村长，村庄安危的直接责任人。外表粗犷，内心沉稳。见过旅行者之后有所察觉，认为对方并非普通流浪者。他了解村庄现状，但不掌握神殿历史深层信息，会将玩家引导至艾尔文。

**对话分支**

| 选项 | 效果 |
|------|------|
| 村子最近出了什么事？ | 揭露矿洞失踪、神殿奇光信息 |
| 我能帮上什么忙吗？ | 推荐玩家去找艾尔文 |
| 我会调查这件事的 | 触发感谢节点，推荐去找艾尔文 |
| 没事了 | 关闭对话 |

---

### 杂货商 梅娜

```
位置：晨曦村·杂货铺（village_east 节点）
ID：grocer
```

**背景**
杂货铺的老板娘，热情好客，嘴皮子利索。消息来源广泛但多为道听途说，是村民八卦的集散地。提供平价消耗品，也是玩家了解村庄近况的重要途径。

**对话分支**

| 选项 | 效果 |
|------|------|
| 让我看看你的货 | 打开商店（与客栈共用小卖部） |
| 村子里最近有什么动静？ | 揭露矿洞失踪和地震传言 |
| 没事了 | 关闭对话 |
