## 世界概况

《Lore Well》发生在一片古老的大陆边缘，文明与荒野交织之处。玩家扮演一名身份成谜的旅行者，偶然踏入晨曦村——一个正处于危机边缘的宁静小镇。

这个世界中魔法曾经繁荣，如今已近乎失落。少数遗迹保存着古代文明的痕迹，而神殿遗迹便是其中最危险、也最神秘的一处。

## 历史年表

| 时期 | 事件 |
|------|------|
| **远古时代** | 古代文明建造神殿，作为连通不同维度力量的圣地；祭司在神殿深处进行神秘仪式 |
| **数百年前** | 一位被历史遗忘的法师发现神殿内部封存着危险的存在，施下了强力封印；法师此后失踪，封印的维护知识仅口耳相传，最终只剩长老艾尔文保存着残缺记录 |
| **数年前** | 封印开始出现松动的迹象，幽暗森林的野生生物行为异常，哥布林族群突然在此区域大规模聚集 |
| **数月前** | 晨曦村周边开始出现史莱姆，村民夜间听到远方传来异响，部分商旅路线中断 |
| **游戏开始** | 旅行者（玩家）抵达晨曦村，长老艾尔文将任务托付于此人 |

## 核心谜题：黑暗蔓延

游戏的核心驱动力是一种被称为"黑暗蔓延"的现象。其表现包括：

- 史莱姆等弱小魔物数量激增，出没范围扩大至村庄周边
- 哥布林行为变得有组织性，出现"守卫"神殿的迹象（非本能行为）
- 幽暗森林深处夜间出现紫色光芒（来源指向神殿遗迹方向）
- 森林狼失去天然的领地意识，变得异常攻击性强

**老板娘玛格的消息**（游戏内对话）：  
> "听说森林里最近出现了异常活动，村民们都不敢靠近……"

**长老艾尔文的判断**（游戏内对话）：  
> "黑暗正在蔓延，我们需要一个勇敢的人去探索古老神殿。"

## 晨曦村的现状

晨曦村是一个人口不多的农业村落，主要经济来源是农耕和有限的林业。三个核心人物构成了村庄的社会结构：

```
长老 艾尔文
├── 精神权威，掌握历史知识
├── 与神殿遗迹的渊源最深
└── 游戏主线的推动者

老板娘 玛格
├── 经济枢纽（客栈是信息与物资的交汇点）
├── 村民情绪的晴雨表
└── 向玩家提供安全"锚点"（可休息）

铁匠 托尔
├── 技术资源提供者（武器、护甲）
├── 与外界商旅有联系（获取铁矿石的渠道）
└── 潜在的支线任务触发者
```

村庄的困境不仅在于外部威胁，内部也有裂痕：村民对长老提出"派人去神殿"的建议有所保留，而托尔更倾向于加固防御而非主动出击。玛格则担心客栈生意，希望局势尽快平息。

## 古代神殿的秘密

神殿遗迹是游戏世界观的核心谜点。目前已知的信息（来自代码与对话）：

**确定事实**
- 神殿建于古代文明时期，地面刻有"无人能解的古老文字"
- 内有一个神秘祭坛，艾尔文的影像在此处显现（说明他与神殿有深层连接）
- 哥布林主动守卫神殿——这不符合哥布林的天然习性，暗示有外力驱动

**古代钥匙的意义**
- 刻有神秘符文，由艾尔文持有并传递给玩家
- 其符文与神殿地面的文字属于同一体系
- 目前钥匙的实际使用功能尚未实现（代码中标注为待扩展）

**封印之内是什么？**  
这是游戏尚未揭晓的核心悬念。以下为设计层面的参考假设：

> 神殿内封存的，可能是一位古代精灵的残魂，或是某种超越维度的存在。它并非纯粹邪恶，而是被错误封印的力量——真正的"黑暗蔓延"来自于封印裂解时泄漏的混沌能量，而非封印物本身的意志。

## 叙事扩展方向

以下为基于现有设定可自然延伸的故事方向，供内容扩展参考：

### 短期（可现在实现）
- **铁匠的铁矿石支线**：托尔需要铁矿石，玩家在森林深处找到矿石后，可换取特制武器——建立经济循环
- **玛格的失踪商旅**：客栈等待一队迟迟未归的商队，玩家可在幽暗森林发现线索

### 中期（需新地点）
- **神殿内部**：使用古代钥匙进入，面对 Boss 战，揭开封印之谜
- **村庄危机**：黑暗蔓延升级，某个晚上村庄遭到大规模侵袭，玩家必须保护村民

### 长期（世界扩展）
- **古代文明遗产**：在神殿内部发现远古文明的记录，引出更大的世界地图
- **艾尔文的秘密**：长老对神殿的了解远超他所承认的，他与法师封印事件可能存在直接关联
- **旅行者的身份**：玩家角色并非普通旅行者——符文钥匙在靠近时发出光芒，可能只有特定血统的人才能使用
