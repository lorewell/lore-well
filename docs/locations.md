## 子地图节点系统

每个大地点内部由多个**子地图节点**（SubLocation）组成，通过小地图界面可在节点间移动。

| 术语 | 说明 |
|------|------|
| `startNodeId` | 进入大地点时的默认起始节点 |
| 节点出口（`exits`） | 跨大地点传送，只在边界节点设置 |
| 节点方向（`north/south/east/west`） | 在大地点内相邻节点之间的导航 |
| 节点交互（`interactions`） | 战斗、NPC 对话、商店、拾取物品等 |

节点在 HUD 小地图中以**圆点**显示，当前节点高亮为紫色，可直接点击移动。

## 晨曦村

```
ID：village
startNodeId：village_center
```

**描述**
> 一个宁静的小村庄，被森林和山脉环绕。这里的人们过着平静的生活，但近来却笼罩在不安的阴影之下。

### 子地图节点

```
        [village_north]
        （瀑布·记忆触发）
              │北
[village_west]─┼─[village_east]
（客栈·村长·长老）  （铁匠铺·杂货铺）
              │南
        [village_center]  ← startNode
        （村子中心·史莱姆战斗）
              │南
        [village_south]
        （出口节点）
        exits: forest / temple_ruins / mine_cave
```

| 节点 ID | 节点名称 | 主要交互 |
|---------|---------|---------|
| `village_center` | 村子中心 | 战斗：史莱姆 |
| `village_north` | 瀑布·隐秘处 | NPC：隐秘瀑布（记忆触发） |
| `village_west` | 暮光客栈 | NPC：老板娘玛格（休息/商店）、村长格雷、长老艾尔文 |
| `village_east` | 铁匠铺 | NPC：铁匠托尔（商店）、杂货商梅娜 |
| `village_south` | 村庄南门 | 出口：幽暗森林 / 古代神殿遗迹 / 废弃矿洞 |

## 废弃矿洞

```
ID：mine_cave
startNodeId：mine_entrance
```

**描述**
> 曾经繁荣的矿场如今一片寂静，黑暗的坑道深处传来令人不安的爬动声。

**世界观背景**  
晨曦村的主要铁矿来源。数月前矿工们因深处出现了不明生物而全部撤离，从此矿洞就被废弃。洞穴蜘蛛盘踞其中，其遗留的铁矿石堆是铁匠托尔所需材料的主要来源。

### 子地图节点

```
[mine_entrance] ← startNode
（矿洞入口·洞穴蜘蛛战斗）
exits: village
      │深入
[mine_deep]
（矿洞深处·洞穴蜘蛛战斗）
      │
[mine_ore_vein]
（矿石矿脉·洞穴蜘蛛战斗·铁矿石掉落）
```

| 节点 ID | 节点名称 | 主要交互 |
|---------|---------|---------|
| `mine_entrance` | 矿洞入口 | 战斗：洞穴蜘蛛；出口：晨曦村 |
| `mine_deep` | 矿洞深处 | 战斗：洞穴蜘蛛 |
| `mine_ore_vein` | 矿石矿脉 | 战斗：洞穴蜘蛛（铁矿石掉落率 50%，铁匠委托核心区） |

**难度参考**：建议等级 LV 2–3，洞穴蜘蛛速度最快（SPD 18），先手优势明显  
**任务关联**：铁匠委托（`quest_blacksmith`）— 收集铁矿石
