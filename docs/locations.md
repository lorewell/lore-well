## 晨曦村

```
ID：village
背景图层：bg_village（暗绿色调）
```

**描述**

> 一个宁静的小村庄，被森林和山脉环绕。这里的人们过着平静的生活，但近来却笼罩在不安的阴影之下。

**世界观背景**

晨曦村是游戏的起点，也是主要的"安全区"。村民世代在此生活，与周边森林保持着脆弱的平衡。随着神殿封印松动，史莱姆开始在村庄外围出没，村民们的日常生活受到了前所未有的威胁。村中的长老艾尔文是唯一了解这段历史的人，他的长老之屋是村庄的精神中心。

**可用交互**

| 交互点 | 类型 | 目标 | 说明 |
|--------|------|------|------|
| 暮光客栈 | 🏠 建筑 | NPC：老板娘玛格 | 可休息恢复 HP/MP，获取消息 |
| 铁匠铺 | 🏠 建筑 | NPC：铁匠托尔 | 装备铸造（功能待完善） |
| 长老之屋 | 👤 NPC | 长老艾尔文 | 触发主线任务、获得世界观背景 |
| 附近的史莱姆 | ⚔️ 敌人 | 史莱姆 | 新手战斗练习，40% 掉落生命药水 |

**出口连接**

| 方向 | 目标地点 |
|------|---------|
| 向东·进入森林 | 幽暗森林 |
| 向南·神殿之路 | 古代神殿遗迹 |

**相关任务**

- **长老的委托**：任务起点（与长老对话后激活）
- **森林的威胁**：任务起点（同上）
- **补给短缺**：战斗史莱姆可获得生命药水完成目标

**探索提示**

- 与老板娘玛格对话选择"休息"可完全回复，适合战斗前准备
- 史莱姆是最弱的敌人，推荐作为第一个战斗目标熟悉战斗系统

## 幽暗森林

```
ID：forest
背景图层：bg_forest（深绿色调）
```

**描述**

> 树木高大茂密，阳光几乎无法穿透枝叶。这里潜伏着各种危险的生物。

**世界观背景**

幽暗森林将晨曦村与更深的荒野隔开，是通往神秘地域的必经之路。森林内的哥布林族群在近几个月内急剧增多，商旅道路几乎断绝。有目击者声称，深夜能看到森林中有紫色的光芒一闪而过——方向正是神殿遗迹所在。

**可用交互**

| 交互点 | 类型 | 目标 | 说明 |
|--------|------|------|------|
| 哥布林巡逻队 | ⚔️ 敌人 | 哥布林 | 45 EXP，可掉落铁剑（10%） |
| 森林狼群 | ⚔️ 敌人 | 森林狼 | 70 EXP，可掉落皮甲（15%） |
| 废弃的行囊 | 📦 物品 | 生命药水 | **一次性**拾取，补充物资 |

**出口连接**

| 方向 | 目标地点 |
|------|---------|
| 向西·返回村庄 | 晨曦村 |
| 向北·深入森林 | 森林深处 |

**相关任务**

- **森林的威胁**：需要在此击败哥布林和森林狼
- **长老的委托**：过路地点，前往森林深处取钥匙的必经路
- **补给短缺**：废弃的行囊可直接提供生命药水

**难度参考**

| 敌人 | 建议等级 |
|------|---------|
| 哥布林 | LV 2+ |
| 森林狼 | LV 3+ |

**探索提示**

- 废弃的行囊只能拾取一次，建议在补给耗尽时再返回
- 森林狼速度极快（SPD 16），先发制人很关键

## 森林深处

```
ID：forest_depths
背景图层：bg_forest_deep（近乎全黑的深绿）
```

**描述**

> 几乎没有人敢踏足的地方，空气中弥漫着古老魔法的气息。

**世界观背景**

森林深处是游戏中危险等级最高的区域之一。古老的树木几乎将天空完全遮蔽，残存的魔法结晶散布在地面。据传说，神殿建造之初，祭司们曾在此地设置结界，阻止未经授权者靠近。结界的核心——石台上的匣子——封存着开启神殿大门的古代钥匙，等待着命运选中之人。

**可用交互**

| 交互点 | 类型 | 目标 | 说明 |
|--------|------|------|------|
| 古老的狼灵 | ⚔️ 敌人 | 森林狼（强化版） | 70 EXP，可掉落皮甲（15%） |
| 石台上的匣子 | 📦 物品 | 古代钥匙 | **一次性**拾取，主线关键道具 |

**出口连接**

| 方向 | 目标地点 |
|------|---------|
| 向南·原路返回 | 幽暗森林 |

**相关任务**

- **长老的委托**：拾取古代钥匙是第二个任务目标

**难度参考**

| 敌人 | 建议等级 |
|------|---------|
| 古老的狼灵 | LV 4+ |

**探索提示**

- 进入本区域前确保携带足够的药水，此处无安全休息点
- 古老的狼灵与幽暗森林的森林狼数据相同，但被设定为更具威胁的存在（建议美术/动画强化时差异化处理）
- 古代钥匙拾取后无需返回长老处，直接前往神殿遗迹即可

## 古代神殿遗迹

```
ID：temple_ruins
背景图层：bg_temple（暗紫色调）
```

**描述**

> 被岁月侵蚀的石柱静静伫立，地面刻满了无人能解的古老文字。

**世界观背景**

神殿建造的确切年代已无从考证，但长老艾尔文的记载表明，这里曾是某一古代文明的圣地。神殿内部的祭坛据说可以连通不同维度的力量。数百年前，一位被遗忘的法师在此施下了封印，将某种危险的存在锁在遗迹之中。如今，封印正在瓦解——这正是黑暗蔓延的根源。

**可用交互**

| 交互点 | 类型 | 目标 | 说明 |
|--------|------|------|------|
| 遗迹守卫者 | ⚔️ 敌人 | 哥布林 | 驻守神殿的哥布林，45 EXP |
| 神秘祭坛 | 🏠 建筑 | 长老艾尔文 | 神殿内部，祭坛显现长老的影像 |

**出口连接**

| 方向 | 目标地点 |
|------|---------|
| 向北·返回村庄 | 晨曦村 |

**相关任务**

- **长老的委托**：抵达此地为第三个目标，到达即完成任务

**难度参考**

| 敌人 | 建议等级 |
|------|---------|
| 遗迹守卫者 | LV 2+ |

**探索提示**

- 抵达此地后任务"长老的委托"自动完成，获得 200 EXP + 100 G
- 神秘祭坛触发的是长老影像对话——这是长老艾尔文知晓神殿内情的体现
- **待实现**：古代钥匙应能打开神殿内部的某个入口，此为后续剧情扩展点

**未来扩展（占位设计）**

## 地点扩展指引

新增地点只需在 `src/data/locations.ts` 的 `LOCATIONS` 对象中追加条目，并在 `src/game/scenes/PreloadScene.ts` 中注册背景图层 key。

**示例模板**

```typescript
// src/data/locations.ts
new_location: {
  id: 'new_location',
  name: '新地点名称',
  description: '地点描述文字',
  backgroundKey: 'bg_new_location',   // 需在 PreloadScene 中创建
  exits: ['village'],                   // 可到达的地点 ID
  interactions: [
    { id: 'new_npc', label: '交互标签', type: 'npc', targetId: 'npc_id' },
    { id: 'new_enemy', label: '交互标签', type: 'enemy', targetId: 'enemy_id' },
    { id: 'new_item', label: '交互标签', type: 'item', targetId: 'item_id' },
    { id: 'new_building', label: '交互标签', type: 'building', targetId: 'npc_id' },
  ],
},
```

```typescript
// src/game/scenes/PreloadScene.ts — createPlaceholderTextures()
this.add.rectangle(0, 0, 1, 1, 0x001122).setDepth(-1)
// 或用真实图片：this.load.image('bg_new_location', '/assets/bg_new.jpg')
```

**交互类型说明**

| type | 触发行为 |
|------|---------|
| `npc` | 打开 NPC 对话框 |
| `building` | 打开 NPC 对话框（建筑入口形式） |
| `enemy` | 触发回合制战斗 |
| `item` | 拾取物品到背包（一次性，已消耗的交互被记录在 `consumedInteractions` 中） |
